## 技术说明（后期字幕）

- AVE API：`https://prod.ave-api.com` + `X-API-KEY` 认证
- MiniMax API：`https://api.minimax.chat/v1/chat/completions` + `MiniMax-Text-01` 模型
- 后端：FastAPI + Python 3.11 + uvicorn
- 前端：React 19 + Vite + Tailwind CSS 4 + Framer Motion
- 数学模型：NumPy 实现，8步复合非线性公式
